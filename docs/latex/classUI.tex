{\footnotesize\ttfamily void UI\+::\+Back\+Track\+Menu (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Displays the Back\+Track menu and prompts the user to choose an option. 

This function presents the user with options to perform TSP calculations using different algorithms. The user can select either\+:
\begin{DoxyItemize}
\item A. Calculate TSP with Nearest Neighbour with Backtrack algorithm for incomplete Graphs without Two Opt Optimization.
\item B. Calculate TSP with Nearest Neighbour with Backtrack algorithm for incomplete Graphs with Two Opt Optimization (Beware that this will take 10x more than the previous).
\item C. Go back to the main menu.
\end{DoxyItemize}

Upon user selection, the function performs the corresponding action.

Time Complexity\+: O(n), where n is the size of the vertex set in the graph. \mbox{\Hypertarget{classUI_a9fb619daf6b6c6cea7888622f7d3a40a}\label{classUI_a9fb619daf6b6c6cea7888622f7d3a40a}}

\doxysubsubsection{\texorpdfstring{pathSelector()}{pathSelector()}}
{\footnotesize\ttfamily void UI\+::path\+Selector (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Selects the appropriate action based on the file path loaded. 

If the file path is empty, it prints an error message and returns. Otherwise, it extracts the last word from the file path and determines the action based on this word.

If the last word corresponds to a specific command (e.\+g., \char`\"{}\+Graph1\char`\"{}, \char`\"{}shipping\char`\"{}), it clears the graph, loads the corresponding data set, and performs the associated action.

\begin{DoxyNote}{Note}
This function assumes that the file path has been previously loaded.

Time complexity\+: O(n), where n is the length of the file path. 
\end{DoxyNote}
